## Requirements

1. bower
2. grunt
3. node & npm
4. compass > 1.0.0
5. sass > 3.3.0
6. breakpoint (ruby gem)

## Installation

```
$ gem install sass
$ gem install compass
$ gem install breakpoint
$ bower install
$ npm install
```

## Run the app

```
$ grunt serve
```
